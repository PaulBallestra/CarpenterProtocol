The mod introduces Block H13 map, a neon industrial arena built around fast team combat, custom equipment, a dedicated HUD, music, atmosphere, and a large library of cyber-themed blocks and props for building similar environments.

## What Players Experience

_Carpenter Protocol_ starts from a <span style="color: #e92d20;">portal </span> to Block H13. Entering the portal brings players into a dedicated <span style="color: #60ffed;">laser tag</span> world with a waiting zone, and a match system built for quick team play.

Once enough players are present, the game starts a countdown, assigns everyone to a <span style="color: #e92d20;">red </span> or <span style="color: #60ffed;">blue </span> team, gives each player their team gear, and teleports them into the arena.

*   <span style="color: #60ffed;">4</span> Players <span style="color: #e92d20;">minimum </span> to start (admin command to use if necessary).
*   <span style="color: #e92d20;">50 </span> kills per team to <span style="color: #60ffed;">win </span> the game (or team empty)
*   Infinite timer
*   <span style="color: #60ffed;">One </span> shot, <span style="color: #60ffed;"><span style="color: #e92d20;">one</span> </span> kill
*   Instant respawn
*   Friendly fire off
*   No loadout changes
*   Fast paced gameplay

## Block H13

Block H13 is the <span style="color: #60ffed;">main </span> playable <span style="color: #e92d20;">map </span> included with the mod. It is designed as a <span style="color: #60ffed;">cyberpunk <span style="color: #e92d20;">laser tag</span></span> arena with strong contrast, LED lighting, metal structures, concrete surfaces, industrial props, signs, leds screens, ducts, and utility objects. Some models are still work in progress.

## Atmosphere

_Carpenter Protocol_ includes its own presentation layer for the match:

*   Custom Block H13 weather.
*   A fixed arena mood and lighting setup.
*   Waiting-zone music.
*   Combat music.
*   Laser gun sounds.
*   Countdown and event sounds.
*   Red and blue laser effects.
*   Portal visual effects.

The goal is for the mode to feel like a complete <span style="color: #e92d20;">arena </span> <span style="color: #60ffed;">activity</span>, not only a block pack.

## Building Content

The mod also adds a large cyber-themed construction library. It includes multiples types, furnitures, struturals, items, deco, ...

These blocks can be used to build futuristic city interiors, industrial corridors, arenas, clubs, rooftops, shops, back alleys, control rooms, and neon-lit public spaces. (Note: recipes aren’t set for custom blocks, so you cannot use them in a survival environment, for now…)

## Work In Progress

<span style="color: #e92d20;">Laser tag bots</span> are present as <span style="color: #60ffed;">WIP </span> content. The project includes red and blue bot definitions, team behavior, and bot weapon setup, but bot spawning is currently treated as development/debug content rather than a finished public gameplay feature.

Some development/debug commands also exist in the codebase, but the public experience is focused on the <span style="color: #e92d20;">portal</span>, Block H13, and the <span style="color: #60ffed;">laser tag</span> match flow.

## Additional Notes

_Carpenter Protocol_ is mostly a one-person project.  
  
Aside from the Block H13 map, created by Hockwerk (who made it possible - thanks mate), everything else has been built solo.  
  
This project started from absolutely zero experience, no background in modding, no time spent in Blockbench, no pixel art, no animation… nothing.  
  
Everything you see here has been learned step by step during development, often through trial, error, and a lot of experimentation.  
  
Because of that, some models don’t fully match Hytale’s art style. They reflect different stages of learning rather than a perfectly consistent pipeline, but they’re also part of the journey that shaped the mod into what it is today.  
  
Thanks for who plays it !
